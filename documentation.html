📌 مستندات API مدیریت جداول دیتابیس

این API امکان ایجاد، خواندن، ویرایش و حذف داده‌ها را در دیتابیس فراهم می‌کند.

⚙️ تنظیمات اولیه

ابتدا اطلاعات دیتابیس را در فایل config.php تنظیم کنید:


$host = "localhost";
$dbname = "test_db";
$username = "root";
$password = "";

🔗 نحوه استفاده از API

📌 دریافت اطلاعات یک جدول

متد: GET

http://yourdomain.com/api.php?action=select&table=users

📌 افزودن داده جدید

متد: POST


http://yourdomain.com/api.php?action=insert&table=users
Content-Type: application/x-www-form-urlencoded

name=Ali&email=ali@gmail.com

📌 ویرایش داده موجود

متد: PUT


http://yourdomain.com/api.php?action=update&table=users
Content-Type: application/x-www-form-urlencoded

id=1&name=Reza&email=reza@gmail.com

📌 حذف داده

متد: DELETE

http://yourdomain.com/api.php?action=delete&table=users&id=1

🖥 تست API

برای تست API می‌توانید از ابزارهایی مانند Postman استفاده کنید.

🛠 تکنولوژی‌های استفاده شده

  • PHP (با PDO برای امنیت بیشتر)
  • MySQL
  • HTML, CSS, JavaScript (برای رابط کاربری)
